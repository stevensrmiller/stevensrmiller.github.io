Gadget Class

Base class for all Gadgets. Subclasses can add as many additional flags and states as they wish.

All Gadgets have these members:
MemberDescription
OnConditionChanged A list of ConditionLists that this Gadget will notify when its completed condition changes.
OnUpdate A list of GadgetObservers that this Gadget will notify when any client code interacts with it.
stayCompleted When true, locks the Gadget into its completed state when it is completed for the first time.
hasCompleted Indicates the Gadget has ever entered its completed state. This is public for debugging. Don't change it.
isCompleted Indicates the Gadget is in its completed state. This is public for debugging. Don't change it. For actual production code, use the IsCompleted property instead.
IsCompleted Indicates the Gadget is in its completed state. For clinet code, this is a read-only property. Internally, when it is set, it will always notify all GadgetObservers that have subscribed to the OnUpdate event (and any other objects that have subscribed to it, as well). If the completed state changes when it is set (that is, when a not completed Gadget becomes completed, or a completed Gadget becomes not completed) it will notify all ConditionLists that have subscribed to the OnConditionChanged event (and any other objects that have subscribed to it, as well).
By contract, all Gadgets start in the not completed state.

Interact(int, int) Client code changes the state of a Gadget by indicating which part of the Gadget to change, and by how much. A Gadget can have any internal structure, but many (like the ComboGadget) will be based on a finite set of state variables (in Gadget slang, these are "rotors"), each of which can be in one of a finite set of states. Thus, interacting with a Gadget will usually be in the form of two numbers, indicating which rotor to change and by how much. (To understand the rotor metaphor, think of a bicycle clock with a few rotating numeric wheels. The whole state of the lock is defined by the number chosen for each wheel, or rotor, on the lock.)
